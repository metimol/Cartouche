# Cartouche Autonomous Service

## Файлы для запуска
- `.env.example` - пример файла с переменными окружения (переименуйте в .env и заполните)
- `main.py` - основной файл для запуска автономного сервиса
- `integration_test.py` - скрипт для тестирования интеграции с C# сервисом
- `validation.py` - скрипт для валидации автономности и производительности

## Документация
- `README.md` - руководство по установке, настройке и использованию
- `REPORT.md` - итоговый отчет о разработке и функциональности
- `architecture.md` - описание архитектуры системы

## Основные модули
- `config.py` - конфигурация системы
- `api_client.py` - клиент для взаимодействия с REST API
- `database.py` - модуль для работы с базой данных
- `bot_manager.py` - управление ботами
- `content_generator.py` - генерация контента
- `reaction_engine.py` - обработка реакций на посты
- `monitor.py` - мониторинг новых постов
- `memory.py` - векторное хранилище для памяти ботов

## Зависимости
- `requirements.txt` - список всех необходимых пакетов

## Директории данных
- `data/` - директория для хранения базы данных и векторного хранилища
- `logs/` - директория для хранения логов

## Запуск системы
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` на основе `.env.example`
3. Запустите сервис: `python main.py`

Подробные инструкции см. в README.md
